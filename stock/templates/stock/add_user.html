Role Permissions Guide
Admin

Full access to all features including user management and access control.

Owner

All features except user management. Can manage purchase orders, stock, and view financial data.

Logistics

Can create purchase orders, request stock transfers, and manage commitments. Cannot receive items or edit stock.

Warehouse

Can manage stock, transfers, and receive items. Cannot see financial data or create purchase orders.

Sales

Can manage customer commitments, transfers, and fulfill orders. Limited access to other features.
